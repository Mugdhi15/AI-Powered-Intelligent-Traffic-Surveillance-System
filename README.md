# AI-Powered Intelligent Traffic Surveillance System

## üìå Project Description  
A smart and automated traffic management system integrating **ANPR** (Automatic Number Plate Recognition) and **ATCC** (Automatic Traffic Classification & Control). It performs real-time vehicle detection, counting, classification, number plate extraction, violation detection, and intelligent traffic flow optimization. The system provides a **Flask-based dashboard** for live monitoring, video uploads, analytics, and database retrieval. Additional features include **helmet detection, triple-riding detection, accident detection, signal-jumping detection, heatmaps**, and **dynamic traffic-light switching**, enabling efficient law enforcement, surveillance, parking management, and urban planning.

## üöÄ Core Features  
- **ANPR:** License plate detection & OCR recognition  
- **ATCC:** Vehicle detection, classification, and counting  
- **Violation Detection:** Helmet, triple riding, signal jumping  
- **Safety Features:** Accident detection  
- **Dynamic Traffic Control:** Smart light switching  
- **Analytics:** Heatmaps, real-time monitoring, recorded video analysis  
- **Dashboard:** Video upload, search by plate, vehicle database
